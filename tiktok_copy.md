### 4.3 Reutilisation du meme video avec captions differentes

Poster le **meme video** avec des captions differentes est une strategie TikTok eprouvee. Plan:

1. **Post 1** (mardi 19h) : Caption A (emotionnelle)
2. **Post 2** (dimanche 10h, 4 jours plus tard) : Caption B (curiosite)
3. **Post 3** (semaine suivante, jeudi 12h) : Caption C (POV)
4. **Post 4** (semaine du 18 mai) : Caption D (directe/urgente)
5. **Post 5** (22-23 mai) : Caption E (social proof)

Espacer de minimum 3-4 jours entre chaque repost. L'algorithme TikTok traite chaque publication comme un contenu distinct.

## 6. DUET / STITCH BAIT

### 6.1 Prompt Duet -- Reaction de maman
```
Texte sur la video : "Filmez la reaction de votre mere quand elle scanne le QR code et duet cette video 😭🎐"

Caption : On veut voir les larmes de vos mamans. Duet avec sa reaction 🥺 #fetedesmeres2026 #emogift #reactionmaman
```

**Mecanique** : Les filles filment leur mere en train de decouvrir le message video. Le format duet place la video originale a cote de la reaction = contenu viral organique.

### 6.2 Prompt Stitch -- Challenge emotionnel
```
Texte sur la video : "Stitch avec le cadeau qui a fait pleurer ta mere"

Caption : Le cadeau qui a fait pleurer ma mere : un carillon japonais avec un message video cache 😭 Stitch avec le votre #cadeauquitouche #fetedesmeres2026 #emogift
```

**Mecanique** : Les utilisatrices stitchent en montrant leur propre cadeau emotionnel, creant une chaine de contenu organique. Emogift est positionne comme le "gold standard" du cadeau emotionnel.

### 6.3 Concept de challenge supplementaire
```
Nom : #DefiMessageMaman
Concept : Enregistrez un message video de 15 secondes pour votre mere SANS pleurer. Impossible.
Caption : Essayez d'enregistrer un message pour votre mere sans pleurer. On vous attend. 😭 #DefiMessageMaman #fetedesmeres2026 #emogift
```

Ce challenge fonctionne parce que :
- Il est emotionnellement irresistible (tout le monde craque)
- Il genere du contenu authentique et vulnerable
- Il ne necessite PAS d'acheter le produit pour participer (portee maximale)
- Il fait connaitre le concept du message video personnalise (entonnoir naturel vers le produit)

## 7. A/B TEST PLAN

### 7.1 Les 3 variantes

| | Variante A | Variante B | Variante C |
|---|-----------|-----------|-----------|
| **Format** | Narrative complete 30s | Quick cut 15s | POV texte overlay |
| **Structure** | Intro (fille prepare) > Cadeau > Scan QR > Reaction mere > Fin emotionnelle | Hook 3s ("ce cadeau fait pleurer toutes les mamans") > Cut direct sur reaction mere > CTA | Ecran noir "POV: tu offres un carillon a ta mere" > Texte defilant expliquant le concept > Reaction a la fin |
| **Audio** | Musique piano douce + son naturel | Son tendance TikTok (trending sound) | Voix off narrative + musique |
| **Hook** | Lent, emotionnel, storytelling | Immediat, choc, curiosite | Identification, immersion |
| **CTA** | Sous-titre final "emogift.com" | Texte "Lien en bio - 49EUR" flash a l'ecran | Texte "Et si tu faisais pareil?" + lien bio |
| **Audience ideale** | Femmes 25-35 sensibles au storytelling | Scrolleuses rapides 18-25 | Communaute TikTok native 20-30 |

### 7.2 Protocole de test

**Phase 1 -- Lancement (jours 1-3)**
- Poster les 3 variantes a 48h d'intervalle
- Meme caption (Option A emotionnelle) pour isoler la variable video
- Meme horaire (mardi/mercredi 19h)

**Phase 2 -- Mesure (jour 4-7)**
- Laisser l'algorithme distribuer pendant 7 jours
- NE PAS booster pendant la phase de test organique

**Phase 3 -- Iteration (jour 8+)**
- Identifier le gagnant
- Decliner le format gagnant avec les 4 autres captions
- Booster (Spark Ads) uniquement le meilleur combo video + caption

### 7.3 Metriques a suivre (par ordre de priorite)

| Metrique | Pourquoi | Seuil de succes |
|----------|----------|-----------------|
| **Taux de completion (watch time)** | Indique si le hook fonctionne et si l'histoire captive | >60% de completion sur 30s, >80% sur 15s |
| **Partages** | Signal viral #1 sur TikTok, "je veux que ma fille voit ca" | >2% du nombre de vues |
| **Sauvegardes** | Intention d'achat ("je garde pour plus tard") | >1.5% du nombre de vues |
| **Clics lien bio** | Conversion directe vers le site | CTR >0.8% |
| **Commentaires** | Engagement communautaire, alimentation algo | >0.5% du nombre de vues |
| **Ratio partages/likes** | Un ratio eleve = contenu viral (les gens partagent plus qu'ils ne likent) | Ratio >0.3 |

### 7.4 Decision framework

```
SI Variante B (15s) gagne en watch time ET clics :
    -> Privilegier le format court pour les ads payantes
    -> Garder Variante A (30s) pour le contenu organique storytelling

SI Variante A (30s) gagne en partages ET sauvegardes :
    -> Le storytelling resonne, investir dans plus de contenu narratif
    -> Utiliser Variante B pour le retargeting (rappel rapide)

SI Variante C (POV) gagne en commentaires ET duets :
    -> Le format communautaire fonctionne, lancer le #DefiMessageMaman
    -> Creer plus de contenu POV participatif
```
